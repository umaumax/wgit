# git wget manager

## required
* `git-sub-wget` command
	* [umaumax/git\-sub\-wget]( https://github.com/umaumax/git-sub-wget )

## how to install
```
wget https://raw.githubusercontent.com/umaumax/diff-filter/master/wgit.sh
mv wgit.sh wgit
chmod u+x wgit
mv wgit <path>
```

## how to use
```
# install
wgit
# help
wgit -h
```

## setting file
`~/.wgit`
`./.wgit`

```
cd ~/local/bin
```
